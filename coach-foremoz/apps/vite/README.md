# Coach Web Mockup

Mockup front-end untuk `coach.foremoz.com` dengan flow:
- signup
- signin
- onboarding
- dashboard

Flow dan route guard mengikuti pattern fitness app (session + protected routes).

## Routes

- `/` landing
- `/signup`
- `/signin`
- `/onboarding`
- `/dashboard`

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
