# 6. Transaction Layer

Setiap event dapat memiliki model transaksi sesuai konteks vertical.

## Transaction Models

- ticket
- booking
- subscription
- donation
- sponsorship

## Contoh

- Yoga class: $10 ticket
- Food tour: $40 per person

## Revenue Sharing

Core platform mendukung pembagian nilai antara:
- creator revenue
- host revenue
- platform fee
- sponsor contribution
