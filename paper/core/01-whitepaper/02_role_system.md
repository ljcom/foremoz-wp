# 2. Role System

Foremoz memakai 3 role inti lintas vertical:
- Creator
- Participant
- Host

## Role Mapping

Creator bisa berupa:
- coach
- instructor
- performer
- tour guide

Participant bisa berupa:
- member
- student
- audience
- tourist

Host bisa berupa:
- studio
- venue
- attraction place

## Multi-role Account

Satu account dapat memegang lebih dari satu role.

Contoh:
- Arman sebagai Creator (coach)
- Arman sebagai Participant (runner)
- Arman sebagai Host (studio owner)
