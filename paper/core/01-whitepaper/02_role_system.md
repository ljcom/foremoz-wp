# 2. Role System

Foremoz memakai 3 role inti lintas vertical:
- Creator
- Participant
- Host

A role is contextual to event participation and operating scope, not a permanent identity type.

## Role Mapping

Creator bisa berupa:
- coach
- instructor
- performer
- tour guide

Participant bisa berupa:
- member
- student
- audience
- tourist

Host bisa berupa:
- studio
- venue
- attraction place

## Multi-role Account

Satu account dapat memegang lebih dari satu role.
Satu Passport dapat beroperasi sebagai Creator, Participant, dan Host di event yang berbeda.

Contoh:
- Arman sebagai Creator (coach)
- Arman sebagai Participant (runner)
- Arman sebagai Host (studio owner)
