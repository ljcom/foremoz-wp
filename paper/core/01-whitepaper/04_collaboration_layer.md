# 4. Collaboration Layer

Core platform mendukung kolaborasi lintas actor untuk mengeksekusi event.

## Contoh Kolaborasi

- coach + studio
- instructor + academy
- performer + venue
- guide + attraction

## Contoh Nyata

- Coach Arman mengadakan yoga class di Sunter Gym.
- Studio mengundang coach untuk mengisi class.

## Fitur Kolaborasi

- invitation
- revenue sharing
- role assignment
- access permission
