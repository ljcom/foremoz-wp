# 3. Event Engine

Event Engine adalah jantung platform. Semua aktivitas utama dipetakan sebagai event.

## Contoh Event

- Yoga Class
- Badminton Tournament
- AI Workshop
- Comedy Show
- City Tour

## Struktur Event

- creator
- event
- time
- location
- capacity
- participants

## Event Lifecycle

- create event
- publish event
- participants join
- check-in
- event completed
- feedback
