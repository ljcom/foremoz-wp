# PROJECT_SCOPE.md
Foremoz Home – Scope Definition

## Objective
Dokumen ini menetapkan batas cakupan untuk whitepaper dan implementasi vertikal Home.

## In Scope
- Domain event utama vertikal Home
- Actor, role, dan permission yang relevan dengan vertikal
- Event model (write) dan read model projection
- Integrasi dengan Foremoz Passport (Identity Layer)

## Out of Scope
- Fitur di luar domain inti vertikal Home
- Kustomisasi enterprise per-client di luar blueprint produk
- Perubahan arsitektur platform-level tanpa decision log

## Relation to Home Whitepaper
Semua keputusan platform-level harus merujuk ke:
- paper/core/01-whitepaper/00_summary.md
