# Glossary

- Event OS: sistem operasi digital berbasis event.
- Passport: identity layer lintas industry.
- Industry Vertical: aplikasi domain spesifik di atas core Event OS.
- EventDB: append-only event store.
- Projector: worker yang membangun read model dari event stream.
- Read Model: model query operasional hasil proyeksi event.
- Policy Matrix: daftar capability per role.
