# 10. Interaction Network

Foremoz membentuk interaction network di mana:
- Creator menghasilkan experiences
- Participant membangun history dan identity
- Host menyediakan infrastruktur
- Sponsor mengamplifikasi dan memonetisasi komunitas

Network ini tumbuh antar Passport (actor identity), bukan antar domain role yang terpisah.

## Network Effects

- more creators -> more events
- more events -> more participants
- more participants -> more sponsor value
- more hosts -> more creator enablement

Interaction network ini menjadi mekanisme pertumbuhan jangka panjang platform.

## Gamification Framework (Yukai Chou Octalysis)

Untuk menjaga partisipasi berulang, Foremoz menerapkan gamification berbasis 8 Core Drives dari Octalysis:
- epic meaning: member merasa berkontribusi pada komunitas/tujuan bersama
- accomplishment: progress, streak, level, dan milestone berbasis aktivitas nyata
- empowerment: member dapat eksplorasi format event, challenge, dan strategi personal
- ownership: history, badge, dan reputasi melekat ke Passport sebagai aset identitas
- social influence: follow, referral, leaderboard komunitas, dan kolaborasi antar actor
- scarcity: akses terbatas untuk event/challenge tertentu berbasis kriteria jelas
- unpredictability: seasonal challenge, surprise reward, dan dynamic campaign
- avoidance: reminder streak, expiration benefit, dan nudges saat engagement menurun

Implementasi gamification di Foremoz mengikuti batasan platform:
- game mechanics harus diturunkan dari event data yang tervalidasi (attendance, completion, contribution)
- reward tidak boleh mendorong manipulasi check-in atau spam activity
- reputasi harus dapat diaudit melalui event log dan read model
- desain insentif dievaluasi per vertical agar selaras dengan outcome domain (fitness, learning, tourism, performance)

Dengan pendekatan ini, gamification berfungsi sebagai sistem motivasi terukur yang memperkuat interaction network lintas industri.
