# 10. Interaction Network

Foremoz membentuk interaction network di mana:
- Creator menghasilkan experiences
- Participant membangun history dan identity
- Host menyediakan infrastruktur
- Sponsor mengamplifikasi dan memonetisasi komunitas

Network ini tumbuh antar Passport (actor identity), bukan antar domain role yang terpisah.

## Network Effects

- more creators -> more events
- more events -> more participants
- more participants -> more sponsor value
- more hosts -> more creator enablement

Interaction network ini menjadi mekanisme pertumbuhan jangka panjang platform.
