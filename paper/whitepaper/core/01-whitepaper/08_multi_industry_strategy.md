# Vertical Modules

Core menjelaskan infrastruktur universal.
Vertical Modules hanya menjelaskan domain extension yang unik.

## Active

Recommended first commercialization vertical.

Fokus unik:
- creator-led physical activity
- memberships and coaching operations
- club/community participation
- competition extension (tournament, ranking, team)

## Learning

Fokus unik:
- program and session structure
- materials and assignment distribution
- certification and skill verification
- cohort dynamics

## Performance

Fokus unik:
- lineup and run-of-show
- ticketing tiers
- production coordination
- fan layer and merch flows

## Tourism (Phase 2)

Tourism diposisikan sebagai natural extension setelah core stabil.

Fokus unik:
- guide-led experiences
- group journeys and itinerary complexity
- Tourism Lite entry sebelum full package orchestration

## Creator Events as Cross-Vertical Pattern

Creator-events diposisikan sebagai reusable pattern lintas vertical, bukan vertical terpisah.
