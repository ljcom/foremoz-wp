# WRITING_RULES.md
Foremoz Learning Whitepaper Rules

## Writing Tone
- Ringkas, teknis, terstruktur
- Hindari marketing fluff
- Fokus pada domain model, event model, read model

## Scope Discipline
- Fokus pada learning lifecycle
- Bedakan platform-level concern ke whitepaper general
- Sponsorship tetap terhubung ke framework general

## Terminology
Gunakan istilah konsisten:
- program
- session
- participant
- instructor
- enrollment
- certificate
- passport
- event
- projection

## Output Discipline
Setiap dokumen menjawab:
- masalah domain learning
- event yang dicatat
- read model yang dibutuhkan
- batasan yang tidak dicakup
