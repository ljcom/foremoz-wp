# Glossary

- namespace: Tenant-isolated event domain identifier (`foremoz:fitness:<tenant_id>`).
- chain: Event stream partition inside namespace, either `branch:<branch_id>` or `core`.
- event: Immutable record of a domain action appended to EventDB.
- projection: Process that converts ordered events into query-ready read model rows.
- read model: Query-optimized table/view generated by projection.
- tenant: One gym/studio business account operating in an isolated namespace.
- branch: Physical or operational location under a tenant, mapped to branch chain.
- subscription: Time-bound membership entitlement assigned to a member.
- booking: Reservation for a class slot by a member or guest.
- PT session: Personal training session unit tracked against a PT package balance.
