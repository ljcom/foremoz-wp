# Glossary

- namespace: tenant-isolated event domain (foremoz:performance:<tenant_id>)
- chain: stream partition (branch:<branch_id> | core)
- event: immutable record pada write layer
- projection: transform stream ke read model
- read model: query-optimized tables/views
- tenant: operator account
- performer: actor pembuat experience
- participant: actor yang mengikuti event
- venue: tempat event berlangsung
- passport: identity layer lintas vertical
