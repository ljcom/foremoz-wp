# Foremoz Performance Whitepaper v0.3 - Scope

## Objective

Foremoz Performance berfokus pada operasi vertical performance berbasis event-driven.


## In Scope

- event lifecycle untuk performance
- actor collaboration: performer, participant, venue, sponsor
- booking/attendance/ticketing sesuai domain
- identity linkage ke Passport
- projection-based read model untuk operasional

## Canonical Surface

- \/web (global landing)
- \/a\/<account> (public account)
- role workspace sesuai kebutuhan vertical performance

## Out of Scope

- full ERP modules
- deep accounting
- custom per-client workflow di luar blueprint produk
