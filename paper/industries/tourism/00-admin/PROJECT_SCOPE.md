# PROJECT_SCOPE.md
Foremoz Tourism – Scope Definition

## Objective
Dokumen ini menetapkan batas cakupan untuk whitepaper dan implementasi vertikal Tourism.

## In Scope
- Domain event utama vertikal Tourism
- Actor, role, dan permission yang relevan dengan vertikal
- Event model (write) dan read model projection
- Integrasi dengan Foremoz Passport (Identity Layer)

## Out of Scope
- Fitur di luar domain inti vertikal Tourism
- Kustomisasi enterprise per-client di luar blueprint produk
- Perubahan arsitektur platform-level tanpa decision log

## Relation to General Whitepaper
Semua keputusan platform-level harus merujuk ke:
- paper/general/01-whitepaper/foremoz-event-os-whitepaper-v1.md
