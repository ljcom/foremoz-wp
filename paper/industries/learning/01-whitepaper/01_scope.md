# Foremoz Learning Whitepaper v0.3 - Scope

## Objective

Foremoz Learning berfokus pada operasi vertical learning berbasis event-driven.


## In Scope

- event lifecycle untuk learning
- actor collaboration: performer, participant, venue, sponsor
- booking/attendance/ticketing sesuai domain
- identity linkage ke Passport
- projection-based read model untuk operasional

## Canonical Surface

- \/web (global landing)
- \/a\/<account> (public account)
- role workspace sesuai kebutuhan vertical learning

## Out of Scope

- full ERP modules
- deep accounting
- custom per-client workflow di luar blueprint produk
