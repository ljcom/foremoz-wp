# Foremoz Learning Whitepaper (Industry Draft)

Menggabungkan workshop + course menjadi satu vertical **Learning** menyederhanakan arsitektur dan membuat platform lebih scalable.

Learning mencakup:
- seminar
- workshop
- course
- bootcamp
- mentoring
- training

Semua merupakan variasi dari learning event.

## 1. Model Konsep Learning

Struktur inti:
- Program (Learning Program)
- Sessions (Learning Events)
- Participants

Contoh:
- Program: AI Builder Bootcamp
- Sessions:
  - Intro to AI
  - Prompt Engineering
  - Build AI App
  - Deployment

Setiap session adalah event.

## 2. Tipe Event dalam Learning

Satu sistem menangani banyak tipe, perbedaan utamanya hanya konfigurasi:
- Seminar: AI Trends 2025
- Workshop: Build AI App
- Course: AI Builder Program
- Bootcamp: Data Science Bootcamp
- Mentoring: Startup Mentoring
- Training: Corporate Training

## 3. Actor dalam Learning

Actor utama:
- Instructor
- Participant
- Organizer
- Venue
- Sponsor

Instructor:
- trainer
- speaker
- mentor

Participant:
- student

Organizer:

## 4. Core Features Learning

### Program Management
- create learning program
- define curriculum
- define sessions
- assign instructors

### Session Management
- session schedule
- instructor assignment
- venue / online link
- capacity control

### Participant Management
- enrollment
- waitlist
- attendance
- progress tracking

### Material Distribution
- slides
- documents
- videos
- assignments

### Certification
- certificate of attendance
- certificate of completion
- skill verification

## 5. Passport Integration

Learning sangat cocok dengan Passport.

Passport dapat mencatat:
- courses attended
- skills learned
- certificates
- topics mastered
- learning streak

Contoh Learning Passport:
- AI Builder Course ✓
- Public Speaking Workshop ✓
- Product Strategy Seminar ✓

Ini membentuk learning identity.

## 6. Community Layer

Learning biasanya membangun komunitas.

Fitur:
- cohort group
- discussion
- follow instructor
- follow topic

Contoh:
- AI Builders Cohort 2026

## 7. Monetization

Sumber monetisasi:
- Program fee (course price)
- Session ticket (workshop/seminar ticket)
- Certification (professional certificate)
- Sponsor (education partner, tools sponsor)
- Content (recorded class, premium material)
